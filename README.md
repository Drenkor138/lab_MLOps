# lab_MLOps

**Лабораторная работа: Разработка и контейнеризация ML-сервиса на FastAPI**

**Цель работы**
Научиться:
- разворачивать и документировать REST API для машинного обучения с помощью FastAPI;
- использовать pre-commit hooks для автоматической проверки качества кода;
- писать тесты для проверки работы API и модели;
- собирать и запускать приложение в Docker.
